# 南京故事

> 标签：  
LPL夏季赛总决赛  
RNG VS IG  
九毛九  

正文：

回想下第一次去南京，那还是因为英雄联盟的缘故，LPL夏季赛总决赛将在南京青奥体育公园举行。秉持着对英雄联盟最后的热爱，果断拉了小伙伴入坑，选座买票付款成功，然后就开始期待在现场看比赛的感觉。
不得不说，因为准备的不够充分，再加上南京的地主也不够给力，@王浩，这次旅行除了比赛看的很过瘾，吃的很爽之外很难再挑出什么值得回忆的地方了。以下除外：
大半夜的骑小黄车狂奔；
秦淮河步行街网吧打游戏；
沙海，爱屋及乌
比赛可以拿出来单说：
九毛九吃的是真的爽，希望不是人生若只如初见。
南京，来日方长。
